MemeFlash by MemeCorp — v1.0.13
Configuration (1 seule fois)
1
Créer un bot Discord
Ouvre discord.com/developers → New Application → nomme-le → onglet Bot :
  • Clique Reset Token → copie le token
  • Active Message Content Intent (obligatoire !)
  • Active aussi Server Members Intent
2
Ajouter le bot à ton serveur
Onglet OAuth2 → URL Generator :
Coche bot, puis Read Messages/View Channels + Read Message History.
Ouvre le lien généré et ajoute le bot à ton serveur.
3
Activer le mode développeur Discord
Discord → Paramètres → Avancé → Mode développeur ✓
Ensuite clic droit sur le salon → Copier l'identifiant = Channel ID.
4
Connecter dans MemeFlash
Onglet Compte → colle le Token et le Channel ID → clique Connecter.
Le point vert confirme la connexion.
Utilisation
▶
Envoyer un mème
Dans le salon Discord configuré, ping l'utilisateur + envoie un média.
Ex : @Pseudo + image / lien TikTok / lien Twitter…
@everyone fonctionne aussi pour tout le monde.
