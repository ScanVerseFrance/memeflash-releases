Configuration (1 fois)
1
Créer un bot Discord
Ouvre discord.com/developers → New Application → onglet Bot :
  • Clique Reset Token → copie
  • Active Message Content Intent (obligatoire)
  • Active Server Members Intent
2
Ajouter le bot au serveur
Onglet OAuth2 → URL Generator : coche bot puis Read Messages + Add Reactions.
3
Récupérer le Channel ID
Discord → Paramètres → Avancé → Mode développeur, puis clic droit sur un salon → Copier l'identifiant.
4
Configuration OBS (optionnel)
Active le serveur dans Réglages, puis dans OBS : + Source → Navigateur, colle http://127.0.0.1:7777/. Largeur/hauteur = celles de ton stream.
